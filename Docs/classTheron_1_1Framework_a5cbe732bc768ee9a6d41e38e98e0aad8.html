uint32_t Theron::Framework::GetCounterValue ( const Counter  counter  )  const

Gets the current value of a specified event counter.

Each Framework maintains a set of internal event counters. This method gets the current value of a specific counter, aggregated over all worker threads.

Parameters:
counter One of several values of an enumerated type identifying the available counters.
Returns:
Current value of the counter at the time of the call.
