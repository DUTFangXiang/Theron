#define THERON_FAIL_MSG ( msg   ) 

Theron error macro.

This define is defined automatically if not predefined by the user. When automatically defined, it is defined differently depending on the value of THERON_ENABLE_ASSERTS. When THERON_ENABLE_ASSERTS evaluates to non-zero, THERON_FAIL_MSG is defined as an unconditional call to Theron::Detail::Fail with the provided message parameter. Otherwise, it is defined away to nothing.

The default definition can be overridden by defining it globally in the build. If so then it must always be defined before inclusion of any Theron headers, in every compilation unit.
