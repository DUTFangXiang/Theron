#define THERON_GCC   0

Enables the use of GCC-specific features.

This define is defined automatically if not predefined by the user. When automatically defined, it is defined as 1 if a version of the GCC compiler is detected, and 0 otherwise.

The default definition can be overridden by defining it globally in the build - either via the makefile command line options, on the GCC command line using -D, or in the project preprocessor settings in Visual Studio.
